# RAG Конструктор по Блуму

## Назначение
Проект для конструирования и проверки RAG-пайплайнов в логике таксономии Блума.

## Дорожная карта
[40 шагов](docs/ROADMAP.md)

## Структура
- backend/ — будущий Python/FastAPI бэкенд
- frontend/ — будущий Next.js фронтенд
- docs/ — документация проекта
- scripts/ — вспомогательные скрипты

git init
