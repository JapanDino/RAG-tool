# Contributing

## Требования
- Python 3.11+
- Установлен pre-commit

## Ветвление
- Вся разработка в ветках `feature/*`

## Коммиты
- Conventional Commits: `feat`, `fix`, `docs`, `chore`, `refactor`, `test`
- Кратко и по делу, одна цель на коммит

## Pull Requests
- Небольшие и сфокусированные изменения
- Описание цели, контекста и шага проверки
- Минимум одно ревью перед слиянием

## Код-стайл
- Автоформатирование: black
- Сортировка импортов: isort
- Линтинг: flake8
- Запуск: `pre-commit run --all-files`
